홈 주요 혜택 교육 과정

1년간의 체계적인 커리큘럼

기초부터 실전 프로젝트까지 현업 인재로 거듭나는 단계별 로드맵입니다.

1학기: Coding Track (기반 다지기)

"탄탄한 SW 기본기와 문제해결 능력 배양"

알고리즘 직중 훈련을 통해 컴퓨팅 사고력을 확장하고, Web Front-End와 Back-End 프레임워크 핵심 기술을 학습하여 독자적인 풀스택 웹 애플리케이션을 구현할 수 있는 기본기를 완성합니다.

2학기: Project Track (심화 역량)

"현업 트렌드 기술 기반 고도화 관통 프로젝트 수행"

자기주도적 팀 프로젝트를 중심으로 진행됩니다. 인공지능(AI), 사물인터넷(IoT), 빅데이터(Big Data), 블록체인 등 최신 도메인 기술을 결합하여 실제 서비스 런칭 수준의 아키텍처를 설계하고 협업 인프라(CI/CD)를 경험합니다.
